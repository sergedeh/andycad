Copying and Pasting

Copy

1.     Select the object or group of objects you wish to copy.

2.     From the Edit menu select Copy  or

3     On the Standard toolbar click.

 

Paste

This operation will  paste the copied object to any position you select using the mouse within the drawing area.

1.     From the Edit menu select Paste or

2.     On the Standard toolbar click.

4.     You can paste multiple copies of the copied object.

5.     To leave the paste mode, on the drawing bar click.

 Note: You must copy object before applying the paste function.
