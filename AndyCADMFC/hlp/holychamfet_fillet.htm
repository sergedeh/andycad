Create Chamfer/Fillet

To create Fillet

  1. Select the object  you wish to Fillet
  2. In the modification toolbar, click
  3. A Chamfer/Fillet length dialog box pops up, enter the desired radius and press "Enter"
  4. Click on the first segment and second segment you wish to  to Fillet.

To create Chamfer

  1. Select the object you wish to Chamfer

  3. A dialog box as above pops up, enter the desired radius and press "Enter"
  4. Click on the first segment and second segment you wish to Chamfer.
