Installing AndyCAD

1. Download AndyCAD  and save file on local disk. 

2. Locate AndyCAD setup file with extension ".exe"

3. Double click  "setup.exe" follow the installation guide  to install AndyCAD on your computer.

4.Runing AndyCAD for the first time. If "launch AndyCAD" was checked during installation the program is launched automatically at the end of installation.

5. Otherwise  and in subsequent times double click AndyCAD shortcut  on your Desktop or Taskbar go to 6.. 

6. Start->Program->AndyCAD->AndyCAD.
