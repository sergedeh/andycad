To Polar copy an object

To polar copy an object with angular rotation

  1. Select the object to copy
  2. On the drawing toolbar, click on the rotation button
  3. A cross marker appears indicating the rotation center of the object
  4. If you wish to displace the center of rotation, click on the cross marker, holding down the left mouse button drag the cross marker to desired center of rotation and release the mouse.
  5. In the transformation toolbar, click polar copy button with angular rotation
  6. A dialog box pops up, enter the rotation angle and the number of copy or copies and click OK or press "Enter"

 

                         

Note:

Polar copy does not rotate the copied object.

if objects are copied without displacing the center of rotation, the objects are copied in the same location as the original object. you may select each to displace as desired.
