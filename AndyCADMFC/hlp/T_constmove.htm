To restraint move

Use this function to move an object around a restricted area you define.

  1. Select the object you wish to move
  2. In the transformation toolbar, click on  the restraint move button
  3. Select the line, curve, segment of curve or object  you wish to use as a reference or boundary
  4. Select the object you wish to move your object and move as desired.

 

Note: the object will only move along the selected boundary.
