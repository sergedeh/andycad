Menu

AndyCAD menu just below the Title bar. The functions find in the menu are also available through the Toolbars and pop-up  menu.

File menu commands

The File menu offers the following commands:

New Create a new document
Open Open a document
Save Save the opened document using the current file name
Save As Save document as (*.deh) or (*.bmp) with a selected file name
Import Import a document to the current file
Page Setup Select the dimension of the working page
Print preview Display the document on the screen as it would appear printed
Print Print the page
Print Setup Select a printer and printer settings
Exit Exit AndyCAD

Edit menu commands

The Edit menu offers the following commands:

Undo Reverse previous operation command
Redo Redo the last reverse operation command
Cut Delete Data from the document and move it to the private clipboard
Copy Copy data from the document to the clipboard
Delete Delete data from the document
Select Select an entity in the document

View menu commands

The View menu offers the following commands:

Toolbar Turn on or off the selected toolbar
Status bar Turn off or on the status bar
Zoom In Enlarge the page by a certain factor
Zoom Out Reduce the page by a certain factor
Fit in window Enlarge or reduce the page to fit the screen window

 

Draw menu commands

The Draw menu offers the following commands:

Line Draws line or polyline:
Orthogonal line Draws perpendicular line.
Rectangle Draws rectangle
Spline Draws Spline and curves
Perpendicular Line Draws line perpendicular to another segment or line
Circle Draws circle, the first point is the center
Circle 2 points Draws circle passing through 2 points
Circle 3 points Draws circle passing through 3 points
Arc Draws an arc, the first point is the center of the arc. 
Arc 2 points Draws arc passing through two points. 
Arc 3 points Draws arc passing through two points. 
Text Draws text

Dimension menu
Horizontal Draws horizontal dimension
Vertical  Draws vertical dimension
Angled inner arrow  Draws angled inner arrow dimension
Angled outer Arrow Draw angled outer arrow dimension
Horizontal  Outer Arrow  Draw horizontal dimension with outer arrow
Vertical Outer Arrow  Draws vertical dimension with outer arrow
Angular  Draws angular dimension
Leader  or annotation Draws leader  or annotation
Diameter Draws diameter (applies to circles)
 

Transform Menu

Mirror Mirror-copy an object.
Mirror (no copy) Mirrors with copy
Symmetric Copy I Copies selected object symmetrically displacing the object to the symmetric position or deleting  the original object and maintaining the mirrored copy.
Symmetric Copy II Copies object symmetrically maintaining the mirrored and original copy .
Angular Array Copies selected object circularly  given the angle and number of copies.
Polar Copy Polar copies  the selected object using the provided parameters.
Rectangular Array Copy Copies within a specified  X and Y axis using he specified parameters.
Parallel Copy Creates a parallel Copy of  selected object from center
Offset Select an object to copy, number of copies and specify offset.
Flip Horizontally Rotate an object horizontally at 90 degrees
Flip vertically Rotate an object vertically at 90 degrees
Slide Button The slide button allow you to move an object centered on  another  object.
Add node Adds node to  a selected object.
Delete Node Deletes  a nodes from a selected object.
Break Break curve or line.
Close curve Close selected figure.
Fillet Select the first segment and second segment to create a fillet.
Chamfer Enter the radius or length. Clicking on the first segment  and  second segment  creates  chamfer on a selected object.
Trim Trims  an object . Selecting the trimming boundary or trimming object, then select the section of the object you wish to trim. 

 

Arrange menu

Group   Group selected objects.
Ungroup  Ungroup selected object (must be previously grouped).
Send front  Send selected object or group of objects to the fore front.
Send back  Send selected object to the background.
Bring forward   Bring selected object one step forward.
Send backward   Send selected object backward.

 

Line Style Menu

Solid Change selected object line style to solid.
Dash Change selected object  line style to dash
Dash dot Change selected object  line style to dash dot
Dash dot dot Change selected object  line style to dash
Line width Normal: Normal line width style
Thick: Select to apply thick width line style.
Thin: Select to apply thin line style
Custom  Select to apply thick line style
