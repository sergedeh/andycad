ToolBar 

Toolbars find in  AndyCAD includes:

Standard Toolbar

 

Drawing Toolbar

　

   

Line /Poly Line: Draw a  2 point line( begin , end), Poly line or an angled line.
    Rectangle: Draws a rectangle.
    Orthogonal line: Draw square lines, vertical,  horizontal or continuos line.
    Cubic B-Spline: Draw a Cubic B-Spline.
    Text : Draw text. 
    Select: Select an object or group of objects.
    Rotate: Rotate an object freely or dynamically  clicking on the  drop down arrow beside pops out the  transformation toolbar.
    Perpendicular line: Draw a perpendicluar line .
    Tangent Line: Draw a tangent line to a circle or an arc.
    Circle: Draw a circle from center.  Click on the drop down arrow beside  pops out the circle toolbar.
 
Circle, Ellipse and Arc Toolbar    

   

Circle from center: Draw circle from center　
    Circle 3 point(1st point, 2nd point and 3rd point):  Draw circle with first point, second point, and 3rd point.　
    Circle 2 point(begin, end): Draw circle with two points.　
    Ellipse: Draw an ellipse specifing first point, second and third point. 　

   

Arc (center, begin, end):  Draw an arc from center specifiying begin and end point.

   

Arc(1st point, 2nd point and 3rd point): Draw an arc specifying first, second  and third point.　

   

Arc 2 points(Semi circle):Draw a semi circle or two point circle specifying  begin and end point.

　

Editing Toolbar
　
    Add node:  Adds node to  a selected object.
Delete Node: Deletes  a nodes from a selected object.
    Break:  Break curve or line.
   Close curve: Close selected figure.
Group: Group selected objects.
   Ungroup: Ungroup selceted object (must be preiously grouped).
   Send front:  Send selected object or group of objects to the fore front.
Send back: Send selected object to the background.
   Bring forward: Bring selected object one step forward.
   Send backward: Send selected object backward.
   Fillet:Select the first segment and second segment to create a fillet.
  Chamfer: Enter the radius or length. Clicking on the first segment  and  second segment  creates  chamfer on a selected object.
Trim:Trims  an object . Selecting the trimming boundrary or trimming object,then select the section of the object you wish to trim. 

Transformation Toolbar  
    Mirror:  Mirror-copy an object.
Mirror (without copy)
Symmetric Copy I:  Copies selected object symetrically displacing the object to the symmetric position or deleting  the original object and maintaining the mirrored copy.
Symmetric Copy II: Copies object symetrically maintaining the mirrored and original copy .
Angular Array: Copies selected object circularly  given the angle and number of copies.
   Polar Copy: Polar copies  the selected object using the provided parameters.
   Rectangular Array Copy:  Copies within a specified  X and Y axis using he specified parameters.
Parrarel Copy: Creates a parrallel Copy of any object from center
   Offset: Select an object to copy, number of copies and specify offset.
   Flip Horizontally: Rotate an object horizontally at 90 degrees
   Filp vertically:Rotate an object vertically at 90 degrees
  Slide Button: The slide button allow you to move an object centered on  another  object.
