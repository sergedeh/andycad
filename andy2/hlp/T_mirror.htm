To Mirror an object

Mirror with copy

  1. Select the object you wish to mirror
  2. In the transformation toolbar, click on mirror with copy button to mirror with copy of the original object
  3. Click on the line, segment of line or object to use as a reference

 

Mirror without copy

Mirror without copy mirrors the selected object  deleting the original object.

  1. Select the object you wish to mirror
  2. In the transformation toolbar, click on  mirror without copy button to mirror without copy of the original object to the mirrored location.
  3. Click on the line or segment of line to use as a reference
