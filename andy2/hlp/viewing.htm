Viewing Objects

Zoom in

This function will increases the view scale of the drawing area.

1.    From the View menu select Zoom in or

2.    On the zooming tool bar click

 

Zoom out

This function will decreases the view scale of the drawing area.

1.    From the View menu select Zoom in or
