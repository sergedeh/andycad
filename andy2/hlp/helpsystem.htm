How to Use Help


AndyCAD provides an online help to help guide users in accomplishing their task. On-screen tips are also available.

  • To view an on-screen tips point on the desired objects with the mouse.
  • You can access the AndyCAD help file by pressing F1 Function Key: Pressing  F1 function key  starts the help systems displaying  its content.

On the Menu Bar->Help->AndyCAD Help: Clicking on AndyCAD Help from the Help menu  will also start the help system. You may pick the Contents, Index, or Find/search tab to search the help system.
