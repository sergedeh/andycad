File menu commands

The File menu offers the following commands:

New Create a new document
Open Open a document
Save Save the opened document using the current file name
Save As Save document as (*.deh) or (*.bmp) with a selected file name
Import Import a document to the current file
Page Setup Select the dimension of the working page
Print preview Display the document on the screen as it would appear printed
Print Print the page
Print Setup Select a printer and printer conection
Exit Exit AndyCAD
