Drawing  Dimension

Horizontal

Draws horizontal dimension 

1. From the Dimension toolbar click  or

2. From the Dimension toolbar click horizontal

3. Select the first point 

4. Holding the mouse drag to the second point

Vertical 

Draws vertical dimension

1. From the Dimension toolbar click  or

2. From the Dimension toolbar click horizontal

3. Select the first point 

4. Holding the mouse drag to the second point

 

Free Angled inner arrow 

Draws free angled dimension with inner arrow 

1. From the Dimension toolbar click  or

2. From the Dimension toolbar click horizontal

3. Select the first point 

4. Holding the mouse drag to the second point

 

Free angled Dimension with outer Arrow

Draws free angled dimension with outer arrow 

1. From the Dimension toolbar click  or

2. From the Dimension toolbar click horizontal

3. Select the first point 

4. Holding the mouse drag to the second point

 

Horizontal  Outer Arrow 

Draw horizontal dimension with outer arrow

1. From the Dimension toolbar click  or

2. From the Dimension toolbar click inner arrow

3. Select the first point 

4. Holding the mouse drag to the second point

 

Vertical Outer Arrow 

Draws vertical dimension with outer arrow

1. From the Dimension toolbar click  or

2. From the Dimension toolbar click outer arrow

3. Select the first point 

4. Holding the mouse drag to the second point

 

Angular 

Draws angular dimension

1. From the Dimension toolbar click  or

2. From the Dimension toolbar click angular

3. Select the first point on a line or line segment 

4. Click on the second on another line or line segment

Note: On objects you wish to apply angular dimension must to be parallel or on the same axis.
